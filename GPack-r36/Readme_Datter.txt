GPackDatter r9


Dat program to check GPack merged files against a standard dat

Usage: GPackDatter [ops] dat dir_containing_merged_files

[ops]
Quick mode: --mode=quick (default)
            Use roms from .grp without checking integrity of the merged files
Print fully matching roms: --print-fullmatch=yes/no (default yes)
Print unmatched roms: --print-unmatched=yes/no (default yes)
Print name matched roms: --print-namematch=yes/no (default yes)
Print hash matched roms: --print-hashmatch=yes/no (default yes)
Print summary: --print-summary=yes/no (default yes)
Recursively search dir containing grp files: --recurse=yes/no (default yes)

Redirect console output to file with > for log
